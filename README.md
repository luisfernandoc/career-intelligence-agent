# Career Intelligence Agent

AI-powered assistant that helps professionals:

- Analyze job opportunities
- Identify skill gaps
- Generate interview questions
- Prepare study plans

Current Features:

- FastAPI API
- Groq integration
- Multi-provider LLM architecture
- Job analysis
- Interview question generation
